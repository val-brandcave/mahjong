## Topics Covered

### 1. Second Charleston (Optional)
- Happens after First Charleston if all players agree
- Same three passes but REVERSED direction
- Left ‚Üí Across ‚Üí Right (opposite of RAL)
- Players can vote to stop after First Charleston

### 2. Stopping the Charleston
- After First Charleston, any player can say "Stop"
- If anyone says stop, Second Charleston doesn't happen
- All players must agree to continue
- It's okay to stop - not required to do Second Charleston

### 3. Second Charleston Pattern
- Pass Left (3 times)
- Pass Across (3 times)
- Pass Right (3 times)
- Memory trick: LAR (opposite of RAL)

### 4. Courtesy Pass (Optional)
- Final optional pass ACROSS only
- After Second Charleston (or First if stopped)
- Pass 0, 1, 2, or 3 tiles - your choice!
- Not required - up to each player

### 5. When Charleston Ends
- After Courtesy Pass (if done)
- Or after First Charleston (if stopped)
- Or after Second Charleston (if no Courtesy Pass)
- Game play begins!
