### Screen 2: Pung - Three Identical Tiles
**Text:**
```
What is a Pung?

A Pung is a set of three identical tiles.

Example: 5 Bam, 5 Bam, 5 Bam

**When to declare Pung:**
‚Ä¢ Your hand requires a set of three identical tiles
‚Ä¢ Someone discards the tile you need
‚Ä¢ You already have two of that tile

**How to declare:**
1. Shout "Pung!" (or just "Call!")
2. Take the discarded tile
3. Expose all three tiles face-up
4. Discard a different tile

Note: Most hands with "NNN" on the card can use Pungs.
```

**Interactive:** Practice Pung scenarios

### Screen 3: Kong - Four Identical Tiles
**Text:**
```
What is a Kong?

A Kong is a set of four identical tiles.

Example: Red Dragon, Red Dragon, Red Dragon, Red Dragon

**Two types of Kongs:**

**1. Exposed Kong:**
‚Ä¢ Someone discards the tile you need
‚Ä¢ You have three already
‚Ä¢ Declare "Kong!"
‚Ä¢ Expose all four tiles
‚Ä¢ Draw a replacement tile from end of wall

**2. Concealed Kong:**
‚Ä¢ You draw the fourth tile yourself
‚Ä¢ Keep all four in your rack (concealed)
‚Ä¢ No need to declare unless winning
‚Ä¢ Some hands require concealed Kongs (marked with C)

Kongs are less common - don't worry if you don't use them often!
```

### Screen 5: Edge Case - Multiple Calls
**Text:**
```
What If Two People Call the Same Tile?

Sometimes two players want the same discard!

**Priority Rules:**

1. **Mahjong (winning) always wins**
   ‚Ä¢ If someone calls to win, they get the tile
   ‚Ä¢ Beats all other calls

2. **Otherwise: Closest player in turn order wins**
   ‚Ä¢ Usually next player in rotation
   ‚Ä¢ Or: player who called first (group decides)

**Example:**
‚Ä¢ East discards 5 Dot
‚Ä¢ South calls to win (Mahjong!) - South gets tile ‚úì
‚Ä¢ West and North both called for Pung - too late

**Clear before playing:** Make sure your group agrees on priority rules!
```

```
‚Ä¢ Too many or too few tiles = Dead hand
‚Ä¢ You cannot win this game
‚Ä¢ Continue playing to avoid disrupting others
‚Ä¢ Learn from mistake for next game!

**Prevention:**
‚Ä¢ Count tiles carefully after Charleston
‚Ä¢ Count before exposing (calling)
```

### Screen 7: Edge Case - Disputed Wins
**Text:**
```
Resolving Win Disputes

Someone declares Mahjong but hand doesn't match card?

**Resolution Steps:**
1. All players examine the tiles
2. Check card pattern carefully
3. Verify concealed/exposed requirements met
4. Count tiles (must be exactly 14)

**If hand is INVALID:**
‚Ä¢ Win is denied
‚Ä¢ Player may be declared dead (depends on group rules)
‚Ä¢ Game continues with other players

**If hand is VALID but not obvious:**
‚Ä¢ Show clearly which line on card
‚Ä¢ Explain pattern matching
‚Ä¢ Other players verify

**Tip:** Always be 100% sure before declaring Mahjong!
```
