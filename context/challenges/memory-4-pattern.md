# Challenge 7.4: Pattern Memory

## Challenge Metadata

**Challenge ID:** MEMORY-004
**Category:** Memory Challenges
**Difficulty:** Easy, Medium, Hard
**Duration:** 3-4 minutes
**Unlock:** 100 stars required
**Skills:** Card memorization, pattern internalization, card fluency

## Description

**Objective:** Study card patterns, then recreate them from memory.

**Why:** Internalizing common patterns helps you recognize them instantly during games.

## Mechanics

1. Show 3 card lines (study for 30-60 seconds)
2. Lines hidden
3. User must recreate patterns using tiles
4. Can use notation or actual tiles
5. Score based on accuracy

## Difficulty Levels

**Easy:** Simple patterns (FF 1111 2222), 60-sec study
**Medium:** Complex patterns, 45-sec study
**Hard:** Very similar patterns, 30-sec study

## Scoring

- Each correct pattern: +300
- All tiles correct: +200 bonus per pattern
- Perfect recall (3/3): +2,000

**Max:** 3,500 points

## 3-Star Requirements

⭐ 1/3 patterns perfect
⭐⭐ 2/3 patterns perfect
⭐⭐⭐ 3/3 patterns perfect
