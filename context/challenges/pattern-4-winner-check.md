# Challenge 1.4: Is This a Winner?

## Challenge Metadata

**Challenge ID:** PATTERN-004
**Category:** Pattern Recognition
**Difficulty Levels:** Easy, Medium, Hard
**Duration:** 3-5 minutes
**Unlock:** 50 stars required
**Skills:** Win validation, error detection, rule enforcement

---

## Description

**Objective:** Quickly determine if a claimed winning hand is valid or has errors.

**Why:** Critical skill for validating your own wins and others' wins during gameplay.

---

## Mechanics

### Setup
- 15 hands per session
- Each shows 14 tiles + claimed card line
- User determines: Valid or Invalid
- If invalid, identify the error
- 15 seconds per hand

### Gameplay
1. Hand displayed with card claim
2. User analyzes: Valid Win or Invalid?
3. Select answer
4. If invalid, select error type:
   - Wrong pattern
   - Wrong concealment (C/X error)
   - Tile count wrong
   - Tiles don't match pattern
5. Feedback + explanation
6. Next hand

---

## Difficulty Levels

**Easy:**
- Obviously valid or obviously wrong
- Clear errors (missing tile, wrong pattern)
- Generous time (20 seconds)

**Medium:**
- Subtle mistakes (one tile off, C/X wrong)
- Similar patterns
- Standard time (15 seconds)

**Hard:**
- Very subtle errors
- Nearly identical patterns
- Quick time (10 seconds)

## Error Types

1. **Pattern Mismatch:** Tiles don't form claimed pattern
2. **Concealment Error:** Exposed when should be concealed (or vice versa)
3. **Tile Count Wrong:** Not exactly 14 tiles
4. **Wrong Suit:** Pattern requires specific suit, tiles are wrong suit
5. **Missing Required Tile:** Pattern needs specific tile type (Dragon, Wind, etc.)
