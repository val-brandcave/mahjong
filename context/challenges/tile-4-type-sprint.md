# Challenge 5.4: Tile Type Sprint

## Challenge Metadata

**Challenge ID:** TILE-004
**Category:** Tile Identification
**Difficulty:** Easy, Medium, Hard
**Duration:** 40-80 seconds
**Unlock:** 50 stars required
**Skills:** Category recognition, quick sorting, tile classification

## Description

**Objective:** Categorize 40 tiles as fast as possible - drag to correct category (Suit, Wind, Dragon, Flower, Joker).

**Why:** Instant category recognition helps with quick rack organization and pattern identification.

## Mechanics

1. Tile appears at top
2. Five category bins below
3. Drag tile to correct bin
4. Next tile immediately appears
5. 40 tiles rapid-fire

## Difficulty Levels

**Easy:** Obvious categories, 2 seconds per tile
**Medium:** All tiles, 1.5 seconds per tile
**Hard:** All tiles including tricky ones, 1 second per tile

## Scoring

- Correct categorization: +50
- Speed bonus (under 1 sec): +25
- Perfect run (40/40): +2,000
- Average under 0.8 sec: +500

**Max:** 5,500 points

## 3-Star Requirements

⭐ 32/40 correct (80%)
⭐⭐ 36/40 correct (90%)
⭐⭐⭐ 40/40 correct (100%)
