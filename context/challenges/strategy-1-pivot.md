# Challenge 6.1: Pivot or Persist?

## Challenge Metadata

**Challenge ID:** STRATEGY-001
**Category:** Strategic Decision
**Difficulty:** Easy, Medium, Hard
**Duration:** 4-5 minutes
**Unlock:** 50 stars required
**Skills:** Strategic pivoting, hand evaluation, decision-making

## Description

**Objective:** Decide whether to change hands or continue with current plan.

**Why:** Knowing when to pivot is crucial - stubbornly pursuing impossible hands loses games.

## Mechanics

1. Show current tiles and target hand
2. Show discards (which tiles are "dead")
3. Show game progress (early, mid, late)
4. User decides: Pivot or Continue
5. System evaluates decision quality
6. Explanation of optimal choice
7. 8 scenarios per challenge

## Difficulty Levels

**Easy:** Clear-cut (tiles are dead, must pivot)
**Medium:** Ambiguous situations (could go either way)
**Hard:** Complex game states, tough strategic calls

## Scoring

- Optimal decision: +200
- Acceptable decision: +100
- Poor decision: 0
- Explanation bonus: +50
- All optimal (8/8): +1,000

**Max:** 3,000 points
