# Challenge 6.4: Joker Judgment

## Challenge Metadata

**Challenge ID:** STRATEGY-004
**Category:** Strategic Decision
**Difficulty:** Easy, Medium, Hard
**Duration:** 4-5 minutes
**Unlock:** 100 stars required
**Skills:** Joker optimization, resource management, flexibility

## Description

**Objective:** Decide when to use Jokers for maximum strategic benefit.

**Why:** Jokers are powerful but limited - using them optimally separates good from great players.

## Mechanics

1. Show tiles + Jokers (1-3)
2. Multiple decision points:
   - Use now or save?
   - Which gap to fill?
   - Keep flexibility or commit?
3. System evaluates each decision
4. 6 scenarios with 2-3 decisions each

## Difficulty Levels

**Easy:** Obvious use cases
**Medium:** Multiple reasonable options
**Hard:** Complex optimization puzzles

## Scoring

- Optimal use: +250 per decision
- Good reasoning: +100
- Perfect strategy (all optimal): +1,500

**Max:** 5,000 points

## 3-Star Requirements

⭐ 8/12 optimal decisions
⭐⭐ 10/12 optimal decisions
⭐⭐⭐ 11/12 optimal decisions
