# Challenge 4.1: Card Explorer

## Challenge Metadata

**Challenge ID:** CARD-001
**Category:** Card Reading
**Difficulty:** Easy, Medium, Hard
**Duration:** 3-5 minutes
**Unlock:** Available from start
**Skills:** Card navigation, pattern finding, card familiarity

## Description

**Objective:** Find specific hands on the NML card as quickly as possible based on given criteria.

**Why:** Familiarity with card organization helps you find hands faster during real games.

## Mechanics

1. System requests: "Find a hand with 4 Dragons"
2. User browses card sections
3. Tap to select matching line
4. System validates selection
5. 10 requests per session
6. Time limit: 60 seconds per request

## Difficulty Levels

**Easy:** General requests ("Find hand with Flowers"), 60 seconds per request
**Medium:** Specific requests ("Find 25-point hand with consecutive runs"), 45 seconds
**Hard:** Very specific ("Find concealed hand with quints under 40 points"), 30 seconds

## Scoring

- Found quickly (under 15 sec): +150
- Found (under 30 sec): +100
- Found (under 60 sec): +50
- Timeout: 0 points
- Perfect round (10/10): +1,000

**Max:** 2,500 points

## 3-Star Requirements

⭐ Find 7/10 hands
⭐⭐ Find 9/10 hands, average under 30 sec
⭐⭐⭐ Find 10/10 hands, average under 20 sec
