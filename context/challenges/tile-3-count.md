# Challenge 5.3: Count 'Em Up

## Challenge Metadata

**Challenge ID:** TILE-003
**Category:** Tile Identification
**Difficulty:** Easy, Medium, Hard
**Duration:** 2-3 minutes
**Unlock:** 20 stars required
**Skills:** Quick counting, visual scanning, tile tracking

## Description

**Objective:** Quickly count how many of a specific tile appear in a group.

**Why:** Important skill for tracking tile availability during games.

## Mechanics

1. 20 tiles displayed in random arrangement
2. Question: "How many 3 Dots?"
3. User enters number (0-4 typically)
4. 10 counting challenges
5. Time limit: 15 seconds per count

## Difficulty Levels

**Easy:** Distinct tiles, easy to spot, 0-2 of target tile
**Medium:** Mixed tiles, 0-3 of target tile
**Hard:** Very similar tiles, large total count (15-20 tiles shown), 0-4 of target

## Scoring

- Correct count: +150
- Speed bonus (under 10 sec): +100
- Perfect round (10/10): +1,000
- No errors: +500

**Max:** 3,000 points

## 3-Star Requirements

⭐ 7/10 correct
⭐⭐ 9/10 correct
⭐⭐⭐ 10/10 correct with speed bonuses
