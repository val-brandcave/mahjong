# Challenge 7.1: Discard Memory

## Challenge Metadata

**Challenge ID:** MEMORY-001
**Category:** Memory Challenges
**Difficulty:** Easy, Medium, Hard
**Duration:** 3-4 minutes
**Unlock:** 50 stars required
**Skills:** Discard tracking, memory, tile availability awareness

## Description

**Objective:** Watch tiles being discarded, then answer questions about the discard pile.

**Why:** Remembering discards helps you know which tiles are still available and what opponents might be playing.

## Mechanics

1. Watch 10-15 tiles get discarded (one by one)
2. Discards disappear from view
3. System asks 5 questions:
   - "How many 5 Bams were discarded?"
   - "Was a Red Dragon discarded?"
   - "Which tile was discarded first?"
   - Etc.
4. Score based on correct recall

## Difficulty Levels

**Easy:** 5 tiles discarded, 3 simple questions
**Medium:** 10 tiles discarded, 5 questions
**Hard:** 15 tiles discarded, 8 detailed questions

## Scoring

- Correct answer: +200
- Perfect recall (all correct): +1,000
- Hard mode perfect: +2,000

**Max:** 3,600 points (Hard: 8 × 200 + 2,000)

## 3-Star Requirements

⭐ 60% correct (3/5 Easy, 3/5 Medium, 5/8 Hard)
⭐⭐ 80% correct
⭐⭐⭐ 100% correct (perfect recall)
