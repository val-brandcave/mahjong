# Challenge 2.3: Rapid Fire Tiles

## Challenge Metadata

**Challenge ID:** SPEED-003
**Category:** Speed Challenges
**Difficulty:** Easy, Medium, Hard
**Duration:** 30-60 seconds
**Unlock:** Available from start
**Skills:** Instant tile recognition, reflexes

---

## Description

**Objective:** Identify 30 tiles as fast as humanly possible.

**Why:** Instant tile recognition is fundamental - every second saved adds up in real games.

---

## Mechanics

1. Tile appears on screen
2. 4 answer options displayed
3. Tap correct name
4. Next tile immediately appears
5. 30 tiles total
6. Faster = higher score

---

## Difficulty Levels

**Easy:**
- Common tiles only
- 2 seconds per tile
- Very different options

**Medium:**
- All tiles
- 1.5 seconds per tile
- Similar options (5 Bam vs 5 Crak)

**Hard:**
- All tiles including similar ones
- 1 second per tile
- Very similar options

---

## Scoring

- Correct: +50 points
- Speed bonus (under 1 sec): +25
- Perfect round (30/30): +1,000
- Average under 0.8 sec: +500

**Max:** 3,750 points

---

## 3-Star Requirements

⭐ 24/30 correct (80%)
⭐⭐ 27/30 correct (90%)
⭐⭐⭐ 30/30 correct (100%)
