# Challenge 7.3: Exposed Tile Tracker

## Challenge Metadata

**Challenge ID:** MEMORY-003
**Category:** Memory Challenges
**Difficulty:** Easy, Medium, Hard
**Duration:** 3-4 minutes
**Unlock:** 100 stars required
**Skills:** Opponent tracking, exposed tile memory, pattern recognition

## Description

**Objective:** Remember what opponents have exposed, then answer questions.

**Why:** Tracking opponents' exposed tiles reveals their likely hands and helps you play defensively.

## Mechanics

1. Show 1-3 opponents with exposed tiles
2. Display for 20-40 seconds
3. Tiles hidden
4. Questions about what was exposed:
   - "What did South expose?"
   - "Did West expose any Dragons?"
   - "How many tiles did North expose?"
5. Score based on accuracy

## Difficulty Levels

**Easy:** 1 opponent, simple exposure, 40-sec view
**Medium:** 2 opponents, mixed exposures, 30-sec view
**Hard:** 3 opponents, complex patterns, 20-sec view

## Scoring

- Correct recall: +250 per opponent
- Perfect memory (all opponents): +2,000
- Detailed accuracy: +100 per specific tile

**Max:** 4,000 points

## 3-Star Requirements

⭐ Basic recall (identify main patterns)
⭐⭐ Detailed recall (70% of tiles correct)
⭐⭐⭐ Perfect recall (100% accurate)
