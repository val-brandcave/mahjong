# Challenge 7.2: What Did I Have?

## Challenge Metadata

**Challenge ID:** MEMORY-002
**Category:** Memory Challenges
**Difficulty:** Easy, Medium, Hard
**Duration:** 2-3 minutes
**Unlock:** 50 stars required
**Skills:** Tile memory, visual recall, rack awareness

## Description

**Objective:** Remember your starting tiles and recreate the rack from memory.

**Why:** Good players remember what they had - helps track tile movements and plan strategy.

## Mechanics

1. Display 13 tiles (visible for 15-30 seconds)
2. Tiles disappear
3. User must select tiles from pool to recreate rack
4. Score based on accuracy

## Difficulty Levels

**Easy:** Simple distinct tiles, 30-second view
**Medium:** Mixed tiles, 20-second view
**Hard:** Very similar tiles, 10-second view

## Scoring

- Each correct tile: +100
- Perfect order: +500 bonus
- Perfect recreation (all 13): +1,500

**Max:** 3,300 points

## 3-Star Requirements

⭐ 9/13 correct tiles (70%)
⭐⭐ 11/13 correct (85%)
⭐⭐⭐ 13/13 perfect recreation
