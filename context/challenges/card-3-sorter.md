# Challenge 4.3: Point Value Sorter

## Challenge Metadata

**Challenge ID:** CARD-003
**Category:** Card Reading
**Difficulty:** Easy, Medium, Hard
**Duration:** 2-3 minutes
**Unlock:** 20 stars required
**Skills:** Risk assessment, point value understanding, difficulty evaluation

## Description

**Objective:** Sort 5 card lines from easiest to hardest based on point values.

**Why:** Understanding difficulty helps you choose realistic hands during games.

## Mechanics

1. Show 5 different card lines
2. User drags to order: Easiest → Hardest
3. Based on point values
4. System validates correct ordering
5. 3 sorting puzzles per challenge

## Difficulty Levels

**Easy:** Very different values (25, 50, 75, 100, 150)
**Medium:** Similar values (30, 35, 40, 45, 50)
**Hard:** Same section, subtle differences

## Scoring

- Perfect order: +500 per puzzle
- Mostly correct (1-2 swaps): +250
- Incorrect: 0
- All 3 perfect: +1,000 bonus

**Max:** 2,500 points
