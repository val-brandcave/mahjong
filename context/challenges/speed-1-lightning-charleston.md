# Challenge 2.1: Lightning Charleston

## Challenge Metadata

**Challenge ID:** SPEED-001
**Category:** Speed Challenges
**Difficulty Levels:** Easy, Medium, Hard
**Duration:** 1-3 minutes
**Unlock:** Available from start
**Skills:** Charleston speed, quick decisions, muscle memory

---

## Description

**Objective:** Complete the First Charleston (Right, Across, Left - 9 passes) as fast as possible.

**Why:** Speed is crucial in real games. Faster Charleston = more time for strategic thinking during gameplay.

---

## Mechanics

### Setup
- Start with 13 tiles in rack
- Full First Charleston: 3 passes of each direction
- Timer starts on first tile selection
- Tracks total completion time

## Tips for Speed

1. **Pre-select mentally:** Know what to pass before round starts
2. **Don't overthink:** Trust your instincts
3. **Use patterns:** Always pass similar tile types
4. **Practice rhythm:** Get into a flow
5. **Muscle memory:** Repetition builds speed
