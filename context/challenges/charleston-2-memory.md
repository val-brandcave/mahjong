# Challenge 3.2: Charleston Memory Test

## Challenge Metadata

**Challenge ID:** CHARLESTON-002
**Category:** Charleston Mastery
**Difficulty:** Easy, Medium, Hard
**Duration:** 4-6 minutes
**Unlock:** 20 stars required
**Skills:** Memory, tile tracking, Charleston awareness

---

## Description

**Objective:** Complete full Charleston, then answer memory questions about what you passed and received.

**Why:** Good players remember Charleston moves - helps track tile locations and make strategic decisions.

---

## Mechanics

1. Complete full Charleston (9 passes)
2. System asks 5 memory questions:
   - "What did you pass to South in Round 2?"
   - "What did you receive from West in Pass Across?"
   - "Did you pass any Jokers?"
   - Etc.
3. Select correct tiles from options
4. Scoring based on correct recall

---

## Difficulty Levels

**Easy:** Remember only last pass (3 tiles)
**Medium:** Remember 3 most recent passes
**Hard:** Remember entire Charleston (all 9 passes)

---

## Scoring

- Correct recall: +200 per question
- Perfect recall (5/5): +1,000
- Hard mode perfect: +2,000

**Max:** 3,000 points

---

## 3-Star Requirements

⭐ 3/5 correct
⭐⭐ 4/5 correct
⭐⭐⭐ 5/5 correct
